# Project Management Database System using SQL

A SQL-based Project Management Database System designed to manage projects, tasks, employees, clients, and task assignments efficiently. This project demonstrates database design, data management, and SQL query analysis using real-world project management scenarios.

## Features
- Created a relational database with multiple interconnected tables
- Implemented Primary Key and Foreign Key relationships for data integrity
- Managed projects, employees, clients, tasks, and task assignments
- Performed project tracking and performance analysis using SQL queries
- Used aggregate functions and filtering techniques for insights generation

## Database Tables
- Clients
- Projects
- Employees
- Tasks
- Task Assignments

## SQL Concepts Used
- CREATE DATABASE
- CREATE TABLE
- INSERT INTO
- JOIN
- GROUP BY
- ORDER BY
- HAVING
- Aggregate Functions (COUNT, AVG, SUM)
- Subqueries
- WHERE Clause

## Example Analysis Performed
- Completed tasks per project
- Employees assigned to multiple tasks
- Overdue projects tracking
- Average tasks per project
- Project with highest employee involvement
- Upcoming task deadlines

## Tools & Technologies
- SQL
- MySQL

## Project Outcome
This project demonstrates how SQL databases can be used for:
- Task tracking
- Resource management
- Team performance analysis
- Deadline monitoring
- Better planning and decision-making
